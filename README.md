Реализация сборщиков логов (log collectors) для мониторинга безопасности: чтение и парсинг логов nginx и SSH, преобразование их в унифицированные сырые события и передача в систему нормализации, а также создание генератора демо-данных для тестирования без реальной инфраструктуры.
— Collectors (ингест логов)
 • Реализация сборщиков:
 • nginx/access.log tail + парсинг
 • (опц.) auth.log/SSH tail + парсинг
 • Приведение к “сырому” событию + проброс в нормализацию
 • Генератор демо-логов (чтобы можно было показать без реальной инфраструктуры)
